Setup guide

Step 1 — Create a Google Sheet
Go to sheets.google.com, create a new sheet, rename tabs to each person's name.

Step 2 — Apps Script
In the sheet: Extensions → Apps Script. Paste the provided code. Deploy as web app, execute as Me, access Anyone.

Step 3 — Notifications
Host this file on GitHub Pages (free) to enable push notifications. Local files can't receive them.

Sheet format
Column A = date (YYYY-MM-DD), Column B = veg or meat.
